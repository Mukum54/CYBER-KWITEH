CYBER KWITEH is a file conversion utility that uses WebAssembly to convert files on your device instead of a cloud. 

## ✨ Features

- 🚀 **Fast** - Convert files directly on your device using WebAssembly
- 🔒 **Private** - No file or file size limits, your files never leave your device
- 🎨 **Beautiful** - Futuristic cyberpunk UI with glowing effects
- 📁 **250+ Formats** - Convert images, audio, documents, and video
- ⚙️ **Customizable** - Conversion settings and preferences
- 🌙 **Dark Mode** - Cyberpunk aesthetic with orange-red glow theme

## 🛠️ Tech Stack

- **Frontend**: SvelteKit 5 + TypeScript
- **Styling**: Tailwind CSS 4
- **Build**: Vite + Static Adapter
- **Conversion**: FFmpeg, ImageMagick, Pandoc (WebAssembly)

## 📦 Deployment

### Deploy on Vercel

1. Push to GitHub:
```bash
git init
git add .
git commit -m "Initial CYBER KWITEH commit"
git remote add origin https://github.com/Mukum54/CYBER-KWITEH.git
git push -u origin main
```

2. Deploy on Vercel:
   - Go to [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Framework preset: **SvelteKit**
   - Build Command: `bun run build`
   - Output Directory: `build`
